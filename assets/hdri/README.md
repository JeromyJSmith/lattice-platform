# HDRI Lighting Environments

`.exr` lighting environments for Redshift renders. Not committed — large files, often licensed.

Suggested set (download from Poly Haven or similar CC0 source):

- `studio.exr` — neutral product-shot studio
- `overcast-morning.exr` — soft daylight for early-stage massing
- `noon-clear.exr` — full sun, hard shadows for solar analysis
- `golden-hour.exr` — warm low-angle light for client presentations
- `dusk.exr` — landscape-lighting and outdoor-kitchen scenes

Paths are written to Pixeltable when a render run is recorded (`lattice/execution/evidence`, `tool='c4d.render'`).
