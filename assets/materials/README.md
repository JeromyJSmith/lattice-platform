# Redshift Material Library

PBR materials for Cinema 4D + Redshift renders. Not committed — these are large, licensed, and per-installation.

Standard landscape set ([`landscape/`](landscape/)):

| Material | Files | Use |
|---|---|---|
| Grass | `grass-*.rsmaterial` | Lawn, meadow, ground cover |
| Bark | `bark-*.rsmaterial` | Tree trunks, branches |
| Stone | `stone-*.rsmaterial` | Boulders, retaining walls |
| Water | `water-*.rsmaterial` | Pool, pond, fountain |
| Concrete | `concrete-*.rsmaterial` | Patios, walkways |
| Wood | `wood-*.rsmaterial` | Fencing, decking, outdoor furniture |

`c4d-exporter.py` in [`../../genai/3d-asset-pipeline/`](../../genai/3d-asset-pipeline/) assigns these to generated plant assets by species rules.
