# Install MCP Server to Claude Code CLI

You can also configure the InfraNodus MCP server in Claude Code. Note, that if you run Claude Desktop and have the MCP servers set up there, Claude Code has a different setup and will use the MCP servers that you install via its CLI or provide in its `.claude.json` file.The easiest way to install it is to run the following command in your terminal:

```bash
claude mcp add infranodus -s user \
-- env INFRANODUS_API_KEY=YOUR_INRANODUS_KEY \
	npx -y infranodus-mcp-server
```

Bash

In this case, a configuration will be added to your Claude Code configuration file usually located at `~/Your_User_Name/.claude.json` (your username home directory). You can also add the configuration manually if you open that file, find `mcpServers` section (create it if it doesn't exist) and add the InfraNodus mcp server activation setting:

```json
{
	"mcpServers": {
		"infranodus": {
			"command": "npx",
			"args": [
			  "-y",
			  "infranodus-mcp-server"
			],
			"env": {
			  "INFRANODUS_API_KEY": "YOUR_INFRANODUS_API_KEY_HERE"
			}
	}
}
```

JSON

## InfraNodus

InfraNodus is a text network visualization tool that can be used to build graphs and represent any text as a network. It's been developed by Nodus Labs since 2011 and has been used in many research institutions and enterprises worldwide.

 InfraNodus runs on [NestJs](http://www.nestjs.com/) Node.Js framework, [Prisma](http://www.prisma.io/), PostgreSQL, and [Sigma.Js](http://www.sigmajs.org/) graph visualization library, uses [Textexture text network analysis algorithm](https://noduslabs.com/research/visualization-text-polysingularity-network-analysis/) as well as Cytoscape, Graphology and other libraries. Without the open-source community and StackOverflow our product would be impossible. That's why we offer a range of [open source tools](https://github.com/noduslabs/) that underpin InfraNodus for public use under AGPL licenses.
