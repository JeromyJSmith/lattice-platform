# Goal

Evaluate whether the FRE minimal loop can serve as a reusable proof kernel for
future projects without forcing LATTICE to subordinate itself to new doctrine.

The first milestone is narrower than adoption. It is:

1. ground the evaluation in research, not just copied packet text
2. normalize the source packet
3. create schemas and examples
4. validate them deterministically
5. emit metrics
6. emit repair tasks for blocking failures
7. emit a promotion decision artifact

Success at this stage means the loop is executable, falsifiable, and restartable
inside a bounded LATTICE namespace.
