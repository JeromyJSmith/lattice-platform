# Geometry agent

Plant geometry pipeline: LOD 100 spike → LOD 300 GLB swap, Fragment edits in the ThatOpen viewer, procedural Blender script generation, GLTF export prep.

Reads `lattice/bridge/plant_assets`, writes new asset rows when a species lacks one.

See [`AGENTS.md`](../../AGENTS.md) and the Plant Geometry section of [`meta/FEATURE_BACKLOG.md`](../../meta/FEATURE_BACKLOG.md).
